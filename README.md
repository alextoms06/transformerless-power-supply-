# TRANSFORMERLESS AC to DC Converter PCB

Tools:
- KiCad 9

Features:
- Bridge rectifier (1N4007)
- 7805 voltage regulator
- LED power indicator
- Filter capacitors (RECTANGULAR RADIAL AND DISK FOOTPRINT)
- 1N4001 THT DIODE
- 2 SCREW TERMINALS FOR INPUT AC AND OUTPUT DC 
- RESISTOR AXIAL
Files:
- Schematic
- PCB layout
- Gerber files
